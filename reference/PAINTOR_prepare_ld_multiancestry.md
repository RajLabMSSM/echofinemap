# Prepare multi-ancestry LD files for PAINTOR

Prepare multi-ancestry LD files for PAINTOR

## Usage

``` r
PAINTOR_prepare_ld_multiancestry(
  dat_merged,
  LD_ls,
  locus_dir,
  superpopulation,
  PT_results_path,
  LD_reference = "1KGphase3",
  force_new_LD = FALSE,
  shared_snps_only = TRUE,
  fillNA = 0,
  nThread = 1,
  verbose = TRUE
)
```

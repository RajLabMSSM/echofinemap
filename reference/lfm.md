# List fine-mapping methods

List all fine-mapping methods currently available in echofinemap.

## Usage

``` r
lfm(finemap_methods = NULL, verbose = TRUE)
```

## Arguments

- finemap_methods:

  Fine-mapping methods to run. See lfm for a list of all fine-mapping
  methods currently available.

- verbose:

  Print messages.

## Examples

``` r
if (FALSE) { # \dontrun{
finemap_methods <- echofinemap::lfm()
} # }
```

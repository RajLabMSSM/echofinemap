# Install PolyFun

Check if PolyFun is installed from the GitHub submodule. If not
installed, provides instructions rather than using interactive prompts
that block non-interactive sessions.

## Usage

``` r
POLYFUN_install(force_new = FALSE, verbose = TRUE)
```

## Arguments

- force_new:

  Force a new installation even when PolyFun is already installed.

- verbose:

  Print messages.

## Value

Logical indicating whether PolyFun is installed.

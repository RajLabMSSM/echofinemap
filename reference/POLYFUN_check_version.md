# POLYFUN: check version

Check PolyFun version installed.

## Usage

``` r
POLYFUN_check_version(polyfun = NULL, conda_env = "echoR_mini", verbose = TRUE)
```

## Arguments

- polyfun:

  \[Optional\] Path to PolyFun executable.

- conda_env:

  Conda environment name.

- verbose:

  Print messages.
